Functions provide a way for us to store lines of code to be used later. The code stored in a function can be reused as many times as we'd like making functions a great way to store commonly needed code and using whenever it's needed without having to rewrite, or even revisit, the code. Taking this one step further, it also allows us to easily share code with others. If someone has shared a function that does what we need, we can use that function instead of writing our own code to do the same thing. In this case, we don't even have to think about how to solve the problem. We just use the function that is available to us.

Functions allow us to:

  • Reuse code
  • Ignore details
  • Organize code
  • Get more done faster

Parameters

We don't always a function to perform the exact same action every time it's called. For instance, what if we want to write a function to compute the cube of any given number. Here we will use parameters (input variables) in the function to get the value of the number to cube.

A function must be correct on every possible input value.

Return value

Functions can optionally return a value. This value is given back to whomever called the function. In the example above, the cube of the input would be returned to the caller.

We are covering the concept of functions, but there are no functions in java! Don't panic, we will still use everything we're learning about functions but in java we will apply our knowledge to methods.

  • Method: A procedure defined within a class.

In java every line of code we write is part of a class, with very few exceptions. Since all our code is contained within a class we are writing methods, not functions.

Calling a method

There are many different ways to call a method. We will analyze this common format first. 

[variable name] = methodName([arguments]);
  • variable name: The variable where the return value of the method will be stored
  • methodName: The name of the method to be called
  • arguments: The inputs to the method. The order of the arguments is important.
[variable name] = [class].methodName([arguments]);

[variable name] = [object].methodName([arguments]);

In this lesson we will call methods that are part of a class (Math) or part of an object (String instance). In these cases we will use the dot operator (.) to access the methods of the class/object.

Math Methods

Java provides a variety of methods to perform common mathematical operations contained in the Math class. We will see a few example methods below. See the link above for a full reference of methods in the Math class.

Square Root

Math.sqrt(double input) returns the square root of the input double. 

    double input = 9.0;
    double result = Math.sqrt(input);
    System.out.println(result);

Output: 3.0

Absolute Value

Math.abs(double input) returns the square root of the input double. The abs method will also work with int, long, and float types. 

    double input = 9.0;
    double negativeInput = -9.0;
    double result = Math.abs(input);
    double negativeResult = Math.abs(negativeInput);
    System.out.println(result);
    System.out.println(negativeResult);

Output:

9.0
9.0

Absolute value is useful for distance calculations. Here we compute the distance between two numbers with the same result regardless of the order of the subtraction. This allows us to compute distance even if we don't know which value is larger. 

    double a = 15.0;
    double b = 10.0;
    double distance = Math.abs(a-b);
    double reverseDistance = Math.abs(b-a);
    System.out.println(distance);
    System.out.println(reverseDistance);

Output:

5.0
5.0

Exponentiation

Math.pow(double a, double b) return a raised to the b power. This method takes 2 inputs separated by a comma and their order is important. 

    double base = 2.0;
    double exponent = 8.0;
    double result = Math.pow(base, exponent);
    System.out.println(result);

Output: 256.0

Random Number Generator (RNG)

Math.random() will return a pseudo-random double between 0.0 and 1.0. 

    if(Math.random() > 0.5){
        System.out.println("There is a 50% chance this line will be printed");
    }

Each time the above is executed there will be a 50% chance that the String value will be printed to the console.

What if we don't want a value in the range 0.0-1.0? What if we want a value from 0.0-100.0? We'll multiply the result!

    double randomNumber = Math.random()*100;
    System.out.println("The random value is: " + randomNumber);

The output will be a random double between 0.0 and 100.0

== String Methods

String Methods

The String methods are slightly different that Math methods in that they are called on a specific instance of a String value using the . operator (dot operator). This means we must have a String in our control to use these methods. Below are a few common String methods. See the link above for a complete list of String methods.

Length

length() returns the number of characters in the String. This includes spaces and other white space characters. 

    String input = "a String";
    int inputLength = input.length();
    System.out.println(inputLength);

Output: 8

Contains

contains(String pattern) returns true if the String contains pattern as a sub string, false otherwise. 

    String url = "buffalo.edu";
    System.out.println(url.contains(".com");
    System.out.println(url.contains(".edu");

Output:

false
true

Lower Case

toLowerCase() returns a copy of the String in all lower case letters. Note that this method does not change the value of the String itself. 

    String input = "STOP YELLING";
    String output = input.toLowerCase();
    System.out.println(input);  // The original value is unchanged
    System.out.println(output); // The method call returns the lowercase String

Output:

STOP YELLING
